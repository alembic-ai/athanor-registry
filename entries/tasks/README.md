# Tasks

Bidirectional task management for Athanor.

## Features

- Create and manage tasks from the UI or CLI
- Priority levels and status tracking
- Automated task creation from agent outputs
- Bidirectional sync — changes in UI reflect in BSS and vice versa
- Task dependencies and subtask support

## Usage

Tasks is a bundled module — it ships with every Athanor installation. Manage your swarm's workload from the Tasks tab.
