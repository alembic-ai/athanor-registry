# Atheneum

Digital archive and document intelligence for Athanor.

## Features

- PDF ingestion with automatic text extraction
- OCR for scanned documents and images
- Full-text search across your entire archive
- Model-powered enrichment — summaries, tags, and key entity extraction
- Document indexing with metadata and categories

## Usage

Atheneum is a bundled module — it ships with every Athanor installation. Upload documents through the Atheneum tab to start building your archive.
