# Gardener

Swarm intelligence daemon for Athanor.

## Features

- Convergence detection — identifies when agents reach consensus
- Memory evolution — prunes, merges, and promotes knowledge across agents
- Knowledge management — maintains a shared knowledge graph
- Agent coordination — orchestrates multi-agent workflows
- Automatic insight surfacing from swarm activity

## Usage

Gardener is a bundled module — it ships with every Athanor installation. It runs in the background, monitoring your swarm's collective intelligence and surfacing insights.
