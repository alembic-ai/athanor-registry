# Feed

Curated output stream for your Athanor swarm.

## Features

- Real-time stream of model-generated content
- Text, images, files, and structured insights
- Filtering by source module, type, and priority
- Timeline view with chronological ordering

## Usage

Feed is a bundled module — it ships with every Athanor installation. It aggregates and displays outputs from all active modules and agents.
