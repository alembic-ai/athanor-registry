# Omni-Data Market Engine

Real-time market data pipeline for Athanor.

## Features

- Multi-source data ingestion: CEX (Binance, Coinbase, etc.), DEX, TradFi
- Order flow and orderbook data
- Macro economic indicators
- On-chain analytics and DeFi protocol data
- Options Greeks and derivatives pricing
- Portfolio state tracking and risk calculations
- Chart data broadcasting to other modules
- Historical replay engine for backtesting

## Usage

Omni-Data is a bundled module — it ships with every Athanor installation. Configure your data sources in the module settings to start streaming market data into your BSS pipeline.
