# Notes

Rich markdown editor built into Athanor.

## Features

- Full markdown editing with live preview
- File tree navigation for organizing notes
- Wikilink support (`[[page-name]]`) for interconnected notes
- Template system for quick note creation
- Auto-save and version history

## Usage

Notes is a bundled module — it ships with every Athanor installation. Access it from the modules tab.
