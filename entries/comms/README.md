# Comms

Unified communication hub for Athanor.

## Features

- Internal message bus for inter-module communication
- Telegram bot integration — send and receive messages
- Discord webhook and bot support
- Email notifications (SMTP)
- Custom webhook endpoints
- Conversation threading and routing

## Usage

Comms is a bundled module — it ships with every Athanor installation. Configure integrations in the Comms settings panel.
