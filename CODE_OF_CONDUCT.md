# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone who participates in the Athanor ecosystem.

## Standards

**Expected behavior:**
- Be respectful and constructive in all interactions
- Submit only code you own or have rights to distribute
- Report security vulnerabilities responsibly
- Help others learn and grow in the community

**Unacceptable behavior:**
- Submitting malicious code, backdoors, or data exfiltration
- Harassment, discrimination, or personal attacks
- Publishing others' private information without consent
- Spamming the registry with low-effort or duplicate submissions
- ID squatting — claiming IDs to block others

## Enforcement

Alembic reserves the right to:
- Remove any entry that violates these standards
- Ban repeat offenders from submitting to the registry
- Revoke badges earned through bad-faith participation

## Reporting

Report violations by opening an issue in this repository or contacting the Alembic team directly.
